# 회고

이곳저곳에 전역변수가 전부 다 들어가서 트러블 슈팅이 너무나 어렵습니다.
하지만 이렇게 하지 않으면 절대 못 만들 거 같더라는 생각이 들어서 
함수를 아주 많이 만들어주고~ 그것을 main()안에서 호출하게 해놓았습니다.
함수를 호출할 때 순서가 틀리면 정상적으로 작동하지 않습니다. 특히 draw_grid가 background.fill(0,0,0) 이전에 호출된다면 당연히, 그려진 격자가 검은색 배경으로 칠해질 테니까요
따라서 호출 순서를 맞춰주는 것은 매우 중요합니다
최대한 주석을 이용해서 함수 안에 본문이 어떻게 작동하는가를 남겨놓고 싶었습니다. (나중에라도 혹시나 건들게 될까 싶어서)
테트리스 만드는 과정을 진행하면서, 알게 된 점 그리고 작동이 안 하는 부분을 매번 기록을 했습니다.
